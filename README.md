# neverlose.dll — reverse engineering repository

Репозиторий с результатами статического анализа `neverlose.dll`, сопутствующей конфигурацией инжектора и восстановленными заметками по внутренней структуре модуля.

## Короткое описание

`neverlose.dll` — 32-битная DLL для загрузки в `csgo.exe` через `Extreme Injector`. По статическому анализу внутри есть UI Neverlose, Lua-скрипты, сетевой обмен, DirectX 9 и встроенные ресурсы. Репозиторий содержит результаты разбора и восстановленные заметки, а не оригинальные исходники.

## Структура репозитория

- `analysis/` — PE-профиль, импорты, интересные строки и снимок конфигурации инжектора.
- `configs/` — сохранённые конфигурационные файлы.
- `docs/` — текстовые пояснения по исследованию.
- `src_recovered/` — восстановленные псевдо-исходники и аналитические заглушки.
- `neverlose.dll` — исследуемый бинарный файл.
- `Extreme Injector v3.exe` — внешний инжектор, найденный рядом с DLL.

## Что подтверждено

- `neverlose.dll` — `PE32 DLL`, архитектура `x86`
- точка входа: `RVA 0x306A78`
- экспортируемых функций не обнаружено
- в `settings.xml` целевой процесс указан как `csgo.exe`
- сохранённая конфигурация использует обычную загрузку DLL, без `ManualMap`
- в импортируемых DLL присутствуют `KERNEL32`, `ntdll`, `WS2_32`, `MSWSOCK`, `ADVAPI32`, `CRYPT32`, `d3dx9_43`
- по строкам внутри файла видны блоки `Lua Editor`, `Scripts`, `Documentation`, shared-функции Neverlose и сообщения о потере соединения с серверами

## Важное замечание

Папка `src_recovered/` не содержит оригинальный source code. Это восстановленные аналитические файлы, составленные по:

- PE-структуре
- таблице импортов
- встроенным строкам
- конфигурации инжектора

Если потребуется полноценная декомпиляция, следующий шаг — разбор DLL в `IDA` или `Ghidra`.
