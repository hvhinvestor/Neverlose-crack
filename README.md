# Neverlose-crack
Репозиторий с результатами статического анализа neverlose.dll: PE-структура, импорты, строки, конфиг инжектора и восстановленные псевдо-исходники. Внутри собраны заметки по Lua-подсистеме, сетевому слою, UI и общей схеме загрузки DLL в csgo.exe. Это reverse engineering, а не оригинальный source code.
